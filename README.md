# Expense Tracker Backend

## Features
- User Signup & Login (JWT Authentication)
- Add Expense
- Get Expenses
- Delete Expense
- Total Expense Calculation

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT

## API Endpoints

### Auth
- POST /users/signup
- POST /users/login

### Expenses
- POST /expense/add
- GET /expense/get
- DELETE /expense/delete/:id
- GET /expense/total

## How to Run
1. npm install
2. create .env file
3. npm start
